## ğŸ“‹ Description

**Yeoman One Source** is a centralized web platform designed specifically for United States Coast Guard Yeomen. It serves as a comprehensive resource hub bringing together essential administrative, personnel, pay, travel, and personnel management tools and guidance in one convenient location.

Whether you're a seasoned Yeoman or just starting out, Yeoman One Source provides the information, tools, and support you need to stay informed, stay organized, and succeed in every aspect of your duties.

## ğŸš€ Getting Started

### Local Development

1. **Clone the repository:**

   ```bash
   git clone https://github.com/devpro012888/yeoman-one-source.git
   ```

2. **Navigate to the project directory:**

   ```bash
   cd yeoman-one-source
   ```

3. **Open in browser:**
   - Simply double-click `index.html` to open in your default browser
   - Or use a local server:

     ```bash
     # Using Python 3
     python3 -m http.server 8000

     # Then open http://localhost:8000
     ```

### GitHub Pages Deployment

The site is automatically deployed to GitHub Pages at:
**https://devpro012888.github.io/yeoman-one-source/**
